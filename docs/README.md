## 快速开始

1. 下载修改器：https://pan.quark.cn/s/958659c83f52
2. 解压到游戏目录
3. 先启动游戏，再启动 `PVZRHTools/PVZRHTools.exe`

## 适配版本

当前适配游戏版本：**3.2.1**
