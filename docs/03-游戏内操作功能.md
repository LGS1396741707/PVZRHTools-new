# PVZRHTools 游戏内操作功能说明

## 布阵功能

### 基本参数

| 参数 | 说明 |
|------|------|
| 行 | 目标行号，设为0表示整行 |
| 列 | 目标列号，设为0表示整列 |
| 重复执行次数 | 操作重复执行的次数（1-50） |
| 砸罐子全屏 | 扩大砸罐子的范围 |

### 生成功能

| 功能 | 说明 |
|------|------|
| 生成植物 | 在指定位置生成选定的植物 |
| 生成植物卡片 | 在指定位置生成植物掉落卡片 |
| 斗蛐蛐快速布阵 | 快速生成5个礼盒+5个僵尸礼盒 |
| 生成僵尸 | 在指定位置生成选定的僵尸 |
| 生成魅惑僵尸 | 勾选后生成的僵尸为魅惑状态（友方） |
| 生成物品 | 生成阳光、金币、梯子等物品 |
| 普通陨星 | 生成普通陨星 |
| 充能陨星 | 生成充能陨星 |
| 究极陨星 | 生成究极陨星 |

### 罐子功能

| 功能 | 说明 |
|------|------|
| 生成植物罐子 | 在指定位置生成包含选定植物的罐子 |
| 生成僵尸罐子 | 在指定位置生成包含选定僵尸的罐子 |
| 生成随机罐子 | 生成随机内容的罐子（需配置文件） |

## 游戏状态调整

### 资源修改

| 功能 | 说明 |
|------|------|
| 修改阳光 | 将阳光设置为指定数值 |
| 锁定阳光 | 阳光数量保持不变 |
| 修改钱数 | 将金币设置为指定数值（0-99999） |
| 锁定钱数 | 金币数量保持不变 |

### 游戏控制

| 功能 | 说明 |
|------|------|
| 生成下一波僵尸 | 立即触发下一波僵尸 |
| 暂停出怪 | 暂停僵尸刷新 |
| 僵尸进家不死 | 僵尸进入房屋不会导致游戏失败（仅本局有效） |
| 无限刷新和积分 | 旅行商店/诸神进化无限刷新，水果忍者无限积分 |

### 清除功能

| 功能 | 说明 |
|------|------|
| 清除全部植物 | 移除场上所有植物 |
| 秒杀全部僵尸 | 立即杀死场上所有僵尸 |
| 清除所有冰道 | 移除场上所有冰道 |
| 魅惑全部僵尸 | 将场上所有僵尸转为友方 |
| 清除所有坑洞 | 移除场上所有坑洞 |

### 其他操作

| 功能 | 说明 |
|------|------|
| 修改关卡名称 | 修改当前关卡显示的名称 |
| 显示字幕 | 在游戏中显示自定义字幕 |
| 启动所有小推车 | 启动所有行的小推车 |
| 生成小推车 | 在指定行生成小推车（不要点太快） |
| 直接生成奖杯 | 直接生成过关奖杯 |
| 深渊作弊 | 深渊模式作弊功能 |
| 加载自定义植物数值表 | 加载自定义的植物属性配置 |
| 僵尸全员站桩 | 让所有僵尸停止移动 |

## 阵容码功能

### 植物布阵

| 功能 | 说明 |
|------|------|
| 获取当前阵型代码 | 导出当前场上植物的阵容码 |
| 根据代码布阵 | 根据阵容码布置植物 |
| 布阵时清空场地 | 布阵前先清除场上植物 |
| 使用高数阵容码格式 | 兼容 @高数带我飞 修改器的阵容码格式 |

### 僵尸布阵

| 功能 | 说明 |
|------|------|
| 获取僵尸阵型代码 | 导出当前场上僵尸的阵容码 |
| 根据代码布阵僵尸 | 根据阵容码布置僵尸 |

### 罐子布阵

| 功能 | 说明 |
|------|------|
| 获取罐子阵型代码 | 导出当前场上罐子的阵容码 |
| 根据代码布阵罐子 | 根据阵容码布置罐子 |

### 混合阵容（仅高数型）

| 功能 | 说明 |
|------|------|
| 植僵混合阵容 | 同时包含植物和僵尸的阵容码 |

## 僵尸海功能

| 功能 | 说明 |
|------|------|
| 极限僵尸海 | 高频率生成大量僵尸 |
| 普通僵尸海 | 每波100个僵尸，需在进入关卡前开启 |
| 僵尸类型 | 选择僵尸海中的僵尸类型 |
| 生成间隔帧数 | 设置僵尸生成的间隔 |

> 注意：僵尸海功能在当前版本可能存在问题
