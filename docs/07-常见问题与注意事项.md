# PVZRHTools 常见问题与注意事项

## 常见问题

### Q: 修改器无法连接游戏？

A: 请确保：
1. 先启动游戏，再启动修改器
2. BepInEx 插件已正确安装到游戏目录
3. 游戏版本与修改器版本匹配（当前适配 3.2.1）

### Q: 某些功能不生效？

A: 可能的原因：
1. 部分功能需要在进入关卡前开启
2. 部分功能仅在特定关卡类型中有效
3. 游戏版本更新可能导致部分功能失效

### Q: 游戏崩溃了怎么办？

A: 尝试以下方法：
1. 重启游戏和修改器
2. 关闭可能冲突的功能
3. 检查是否使用了不稳定的功能（如锁定子弹类型）

### Q: 阵容码无法导入？

A: 请检查：
1. 阵容码格式是否正确
2. 是否选择了正确的阵容码格式（高数型/普通型）
3. 僵尸数量是否超过2000（超过会导致崩溃）

### Q: 快捷键不生效？

A: 请确保：
1. 切换为英文输入法
2. 使用组合键而非单键
3. 游戏窗口处于激活状态

## 已知问题

### 僵尸海功能

- 当前版本僵尸海功能可能存在问题
- 开发者表示"不会修"

### 卡槽替换功能

- 存在较多无法修复的 bug
- 仅供娱乐使用
- 重启游戏可能解决问题
- 0号植物（豌豆射手）替换已被禁用

### 锁定子弹类型

- 可能引发各种奇怪的 bug
- 仅供娱乐，不会做针对性修复

### 关卡特性附加

- 排山倒海、种子雨、胆小菇之梦等功能
- 可能与其他功能存在兼容冲突
- 请勿反馈这些功能引发的 bug

## 注意事项

### 安全提示

- 本修改器完全免费且已在 GitHub 开源
- 若遇到付费/要求三连+关注/私信发链接的情况，说明你被骗了
- 请注意隐私和财产安全

### 使用建议

1. 建议在单机模式下使用修改器
2. 使用前建议备份存档
3. 不稳定功能请谨慎使用
4. 遇到问题可尝试重启游戏

### 性能注意

- 植物攻击无间隔等功能可能影响游戏性能
- 植物叠太多可能导致问题
- 高频功能（如僵尸海）可能导致卡顿
