# PVZRHTools 传送带与花园修改器说明

## 传送带植物类型修改

### 功能说明

修改传送带关卡中出现的植物类型。

### 使用方法

1. 在"传送带植物类型"区域添加想要的植物
2. 重新进入关卡使修改生效
3. 传送带将只出现设置的植物类型

### 注意事项

- 修改后需要重新进入关卡才能生效
- 清空列表将恢复默认传送带植物

## 花园修改器

### 功能说明

内置了 @听雨夜荷 的花园修改器 pvzRH-GardenEditor，并适配了动态 ID 和贴图加载。

### 打开方式

点击"花园修改器（打开较慢，别急）"按钮。

### 注意事项

- 花园修改器打开速度较慢，请耐心等待
- 在 3.0.1 版本后花园修改器功能已被删除

## 自定义植物数值表

### 功能说明

加载自定义的植物属性配置文件，修改植物的各项数值。

### 使用方法

1. 准备植物数值配置文件
2. 点击"加载自定义植物数值表"按钮
3. 选择配置文件加载

### 注意事项

- 由于代码限制，无法保证所有条目均能生效
- 部分硬编码的数值可能无法修改

## 深渊作弊

### 功能说明

深渊模式的作弊功能。

### 使用方法

点击"深渊作弊"按钮即可。
