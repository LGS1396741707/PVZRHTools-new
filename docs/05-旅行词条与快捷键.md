# PVZRHTools 旅行词条与快捷键说明

## 旅行词条修改（Tab3）

### 初始植物词条设置

在每局游戏开始时（或生存模式第1轮）自动添加的词条。

- 勾选想要的词条
- 局内修改无效，需要在进入关卡前设置
- 点击"全选"可选中所有词条
- 点击"全不选"可取消所有选择

### 植物词条实时修改

与游戏实时同步的词条设置。

- 可在游戏中任意增删词条
- 修改立即生效
- 仅对本局有效

### 初始僵尸词条设置

在每局游戏开始时自动添加的僵尸词条（Debuff）。

### 僵尸词条实时修改

与游戏实时同步的僵尸词条设置。

## 卡槽替换（Tab4）

> 注意：此功能存在较多 bug，仅供娱乐，重启游戏可能解决问题

| 设置项 | 说明 |
|--------|------|
| 是否应用 | 勾选后该条目生效 |
| 原植物类型 | 要被替换的植物 |
| 新植物类型 | 替换后的植物（设为负数表示不修改） |
| 自定义阳光 | 自定义该卡片的阳光消耗 |
| 自定义冷却时间 | 自定义该卡片的冷却时间 |

> 警告：0号植物（豌豆射手）替换会引发严重问题，已被禁用

## 全局按键绑定（Tab5）

### 设置方法

1. 切换为英文输入法
2. 将鼠标指针放在对应条目的框上
3. 先按下 Ctrl/Shift/Alt
4. 再按下要组合的键位

### 注意事项

- 请不要设置单键，建议使用组合键
- 部分功能需提前填好数据
- 游戏运行时其他软件中下列键位均失效，可能导致冲突
- 按键绑定控件来自 FastHotKeyForWPF，使用时务必切换成英文输入法

## 游戏内按键绑定（Tab6）

### 特点

- 仅支持单键
- 按键仅在游戏内生效
- 不会影响其他软件

### 默认快捷键

| 按键 | 功能 |
|------|------|
| 数字键 3 | 时间减速（0.2倍速） |
| 自定义 | 时间暂停 |
| 自定义 | 显示游戏信息（波数、刷新CD等） |
| 自定义 | 卡槽置顶切换 |
| 自定义 | 图鉴创建植物 |
| 自定义 | 图鉴创建僵尸 |
| 自定义 | 图鉴僵尸魅惑切换 |
| 自定义 | 图鉴创建植物罐子 |
| 自定义 | 图鉴创建僵尸罐子 |
| 自定义 | 随机卡槽切换 |

## 杂项设置（Tab7）

| 功能 | 说明 |
|------|------|
| 关闭时保存现有设置 | 关闭修改器时保存当前设置（数值修改与游戏内词条修改除外） |
| 悬浮窗 | 双击悬浮窗使修改窗口前置，推荐全屏时使用 |
| LICENSE | 查看本项目及所使用开源库的许可协议 |

## 图鉴快捷创建功能

1. 打开游戏内图鉴
2. 点击想要的植物/僵尸
3. 使用快捷键在鼠标位置创建该植物/僵尸

### 支持的图鉴类型

- 植物图鉴（AlmanacCard）
- 僵尸图鉴（AlmanacCardZombie）
- 新版图鉴UI（AlmanacCardUI）
- 植物图鉴控制器（AlmanacPlantCtrl）
