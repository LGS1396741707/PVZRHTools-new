# PVZRHTools 安装与启动说明

## 项目简介

PVZRHTools 是《植物大战僵尸：融合版》的修改器工具，由 @Infinite75 原创，@听雨夜荷 和 @梧萱梦汐X 接手维护。

当前适配游戏版本：3.3.0

## 下载地址

已构建版本：https://pan.quark.cn/s/958659c83f52

## 安装步骤

1. 下载修改器压缩包并解压
2. 将解压后的所有文件复制到游戏根目录，目录结构如下：

```
游戏目录/
├── BepInEx/                     # BepInEx 框架
│   ├── config/                  # 配置文件
│   ├── core/                    # 核心文件
│   ├── plugins/                 # 插件目录
│   │   ├── ToolModBepInEx.dll   # 游戏插件
│   │   └── ToolModData.dll      # 数据定义
│   └── ...
├── PVZRHTools/                  # 修改器程序
│   ├── PVZRHTools.exe           # 修改器主程序
│   └── ...
├── doorstop_config.ini          # Doorstop 配置
├── winhttp.dll                  # Doorstop 代理
├── PlantsVsZombiesRH.exe        # 游戏主程序
└── ...
```

## 启动方式

1. 先启动游戏 `PlantsVsZombiesRH.exe`
2. 游戏启动后，运行 `PVZRHTools/PVZRHTools.exe`
3. 修改器会自动连接游戏

## 语言设置

修改器支持三种语言：
- 简体中文
- English
- Русский (俄语)

在修改器界面右上角的下拉框中切换语言。
