# PVZRHTools 基本特性功能说明

## 常用功能（Tab0）

| 功能 | 说明 |
|------|------|
| 开发者模式 | 启用游戏内置的开发者调试功能 |
| 手套无CD | 手套使用后立即可用，无需等待冷却 |
| 锤子无CD | 锤子使用后立即可用，无需等待冷却 |
| 植物卡槽无CD | 所有植物卡片无冷却时间 |
| 自由种植 | 可在任意位置种植植物，无需满足种植条件 |
| 解锁全部融合配方 | 解锁所有植物融合配方 |

## CD 锁定功能

| 功能 | 说明 |
|------|------|
| 锁定手套CD | 将手套冷却时间锁定为指定数值 |
| 锁定锤子CD | 将锤子冷却时间锁定为指定数值 |
| 两波间最大刷怪CD | 设置两波僵尸之间的最大间隔时间（0-30秒，超过30秒会崩溃） |
| 游戏速度 | 调整游戏运行速度（0-16倍） |

## 关卡特性附加（Tab1.Page4）

| 功能 | 说明 |
|------|------|
| 排山倒海 | 启用排山倒海关卡模式 |
| 种子雨 | 启用种子雨关卡模式 |
| 胆小菇之梦 | 启用胆小菇之梦关卡模式 |

## 场地特性（Tab1.Page5）

| 功能 | 说明 |
|------|------|
| 不生成冰道 | 冰车僵尸不会生成冰道 |
| 取消冰冻特效 | 禁用全屏冰冻特效，但保留冻结僵尸和雪原植物充能效果 |
| 砸罐子修复 | 修复砸罐子关卡问题：重叠罐子只砸一个、小丑/巨人/AOE不破坏罐子 |
| 阳光无上限 | 取消50000阳光存储上限限制 |
| 磁力坚果无限 | 磁力坚果无限吸引子弹，取消100个子弹存储限制 |
| 物品不消失 | 掉落的卡片、肥料、水桶等物品不会消失 |
| 卡槽无初始CD | 游戏开始时卡槽没有初始冷却时间 |
| 小丑盒子不爆 | 小丑僵尸的盒子不会爆炸 |
| 大蒜天 | 每隔一段时间自动对所有僵尸施加大蒜效果 |
| 不生成坑洞 | 不生成地面坑洞 |

## 植物特性（Tab1.Page6）

| 功能 | 说明 |
|------|------|
| 植物攻击无间隔 | 植物射击无间隔，持续攻击 |
| 植物无敌 | 植物不会受到伤害 |
| 魅帝菇无CD | 魅惑帝王菇召唤僵尸无冷却 |
| 地雷无CD | 土豆地雷准备时间极短 |
| 大嘴花无CD | 大嘴花吞噬后立即可再次吞噬 |
| 玉米加农炮无CD | 玉米加农炮发射无冷却 |
| 子弹无限穿透 | 植物子弹可无限穿透僵尸 |
| 窝红温了 | 开发者模式珞瓜（植物叠太多可能导致问题） |
| 超级机枪射手无限开大 | 超级机枪射手可无限使用大招 |
| 植物升级 | 启用植物升级功能 |
| 超桃无CD | 超级桃子无冷却时间 |
| 锁定大麦变植 | 锁定大麦变出的植物类型 |

## 其他特性（Tab1.Page7）

| 功能 | 说明 |
|------|------|
| 水果忍者无限积分 | 水果忍者模式不会扣分 |
| 诸神进化无限刷新 | 诸神进化模式可无限刷新 |
| 旅行随机无限刷新 | 旅行商店可无限刷新 |
| 水果落地自动切 | 水果落地前自动切开 |
| 随机卡槽 | 卡槽植物随机变化 |
| 手套动一排 | 手套移动时整排植物一起移动 |
