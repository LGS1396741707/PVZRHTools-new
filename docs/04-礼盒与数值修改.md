# PVZRHTools 礼盒与数值修改说明

## 礼盒修改（Tab1.Page2）

| 功能 | 说明 |
|------|------|
| 超级礼盒 | 礼盒开出更强力的植物 |
| 盒子僵尸概率出究极僵尸 | 盒子僵尸有概率生成究极僵尸 |
| 礼盒快速打开 | 礼盒立即打开，无需等待动画 |
| 锁定礼盒植物 | 锁定礼盒开出的植物类型为指定植物 |

## 数值修改（Tab1.Page3）

> 提示：将数值设为 -1 可还原为默认值

### 生命值修改

| 功能 | 说明 |
|------|------|
| 植物初始血量修改 | 修改指定植物的初始生命值 |
| 僵尸初始血量修改 | 修改指定僵尸的初始生命值 |
| 一类防具血量修改 | 修改僵尸一类防具（如路障、铁桶）的血量 |
| 二类防具血量修改 | 修改僵尸二类防具的血量 |

### 子弹修改

| 功能 | 说明 |
|------|------|
| 子弹伤害值修改 | 修改指定子弹类型的伤害值 |
| 锁定全部子弹类型 | 将所有植物发射的子弹锁定为指定类型 |

> 注意：
> - 子弹名称保留源代码中的英文命名，若看不懂可以试一下
> - 锁定子弹类型可能引发各种奇怪的 bug，仅供娱乐

## 使用示例

### 修改植物血量

1. 在"植物初始血量修改"下拉框中选择目标植物
2. 在数值框中输入新的血量值
3. 点击"修改"按钮

### 修改僵尸血量

1. 在"僵尸初始血量修改"下拉框中选择目标僵尸
2. 在数值框中输入新的血量值
3. 点击"修改"按钮

### 锁定礼盒植物

1. 勾选"锁定礼盒植物"
2. 在下拉框中选择想要的植物
3. 之后所有礼盒都会开出该植物
